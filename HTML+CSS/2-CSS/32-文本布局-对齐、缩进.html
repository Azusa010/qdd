傍晚的阵雨刚停，玻璃门外还挂着水珠，把街灯晕成一圈圈暖黄。 林野推开便利店门时，风铃叮铃一响，冷气裹着关东煮的香味扑面而来。他习惯性走向最里面的冰柜，想拿一瓶冰咖啡，却在货架拐角停住了脚步。 一个穿校服的女生正踮着脚，够顶层的草莓牛奶。她的发梢还带着一点湿意，侧脸被冷柜的灯光照得格外柔和，手指几次快要碰到，又轻轻滑开。 “我帮你吧。” 林野伸手，轻松取下那盒粉色包装的牛奶。女生回过头，眼睛像刚被雨水洗过一样亮，小声说了句 “谢谢”。 他笑了笑，没再多说，拿起自己的冰咖啡走向收银台。结账时，身后又传来轻轻的脚步声，刚才那个女生抱着牛奶，排在了他后面。 雨已经完全停了。 走出店门，晚风带着湿润的青草气息吹过来。林野刚要打开咖啡，身后忽然有人叫住他。 “那个……” 女生小跑几步追上，把一颗柠檬糖塞进他手里，“刚才真的谢谢你。” 不等他回应，她便红着脸转身跑开，帆布鞋踩过积水的路面，溅起小小的水花。 林野低头看着掌心的糖，包装纸在暮色里闪着微光。 原来一场普通的雨后傍晚，也会捡到这样一小块，轻飘飘的温柔。
