属性 作用 常用值
background-color 设置背景颜色 颜色名称、十六进制、RBB、RGBA
background-image 设置背景图片 url('image.jpg')
background-repeat 设置背景图片是否重复 repeat(默认）、no-repeat、repeat-x、repeat-y
background-position 定位背景图片位置 x y(如center top、或者px、％为单位)
background-attachment 设置背景图片是否随内容滚动 scroll(默认)、fixed
background-size 设置背景图片的尺寸 cover、contain、auto、或者指定宽高
